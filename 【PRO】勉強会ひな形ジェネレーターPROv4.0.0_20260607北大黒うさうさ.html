情報入力 → 9点セットのひな形を生成 → 画面で書き込み編集 → Markdown / テキスト / CSV / PDF で保存 ｜ 北大の金色の並木道を歩く気分で🍂

📖 使い方 — 入力して、かきかきして、書き込んで、保存するまで
1勉強会の情報を入力：左のフォームに勉強会名・日付・時間・講師・テーマなどを記入します（*は必須）。迷ったら「🎁 サンプル投入→出力」で3種のプリセットをワンクリックお試し。
2コマ構成でタイムテーブル自動計算：1行に「名称,分数,種別」で書くと、開始時刻から「10:00–10:20」形式を自動計算して講座資料と台本に反映。終了時刻と合わない場合は⚠️で正直にお知らせします。
3✨生成：黒うさ先生が机でかきかきしたら、9点セット（講座資料・台本・運営ガイド・ヒント・解答・宿題・サポート・チェックリスト・表彰状）のひな形が完成。タブで切り替えられます。
4そのまま書き込み編集：本文エリアをクリックして直接編集できます。編集すると「✏️編集済み」が付き、「↩️ ひな形に戻す」でいつでも初期文面へ。
5保存（4形式）：保存形式で Markdown / テキスト / CSV / PDF を選び、「💾この1枚」または「⬇️9点まとめて」。PDFはブラウザの印刷画面で「PDFに保存」を選ぶ方式で、表彰状だけ金枠の賞状レイアウトになります（🏆欄で受賞者名を差し込み、空欄なら手書き用の下線）。
6選択ダウンロード：チェックで選んで「まとめて1ファイル」or「個別ファイル連続保存」。📌受講生配布セット／📌講師セットのパターンを使うと一発選択。「👀結合プレビュー」で保存前に中身を確認できます。
7🔍検索・置換／🩺記入漏れ検査：9資料を横断して一括置換（原本は保持）。検査ボタンで（日付未定）などの記入漏れや⚠️残りを行番号つきで一覧し、クリックで該当資料へ飛べます。
8残したいときは2段構え：「🗂 いまの状態を一時保存」はこのページを開いている間の退避（最大5件）。長期保存は「📦 ファイルへ保存(.json)」→次回「📂 ファイルから復元」か、.jsonを画面へドラッグ&ドロップで丸ごと復元できます。
💡 大事な約束（ウソつかず）：このツールは端末に何も保存しません。ページを閉じると入力・編集・一時保存は消えます——残したいものは必ず「📦 ファイルへ保存」を。生成物はあくまでひな形（たたき台）なので、実際の進行に合わせて書き込み・調整してから配布してください🐰🥕
